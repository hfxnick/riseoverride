Is This Thing On?

Published on June 5, 2025 by Team @ Override Labs

After a few months of building, rebuilding, patching and testing SCORM downloads at 2 a.m., RiseOverride is officially live in the Chrome Web Store.

We're calling this version 1.0. It’s officially a thing now.

What It Does

RiseOverride gives you an easy way to inject custom CSS (and even JavaScript) into your Articulate Rise SCORM exports. You decide what code to include—we make sure it gets added cleanly, without having to dig into zip files or break anything.

Free users can quickly patch their SCORM files using our manual method—no need to unzip, edit, and repackage files by hand. It’s a huge time-saver that simplifies a tedious process. Pro Mode goes even further by automating the patching entirely, applying your custom code the moment you download your SCORM export. Either way, you’re in control and saving time. This all happens privately and securely on your device.

What’s Free

  • Manual SCORM file patching using our custom built zip patcher
  • A basic editor for CSS (better one coming soon)
  • SVG editor (in progress but shipping soon)
  • All of it runs locally—nothing gets uploaded or tracked (we don't even have a server at the moment)

What’s Pro

Pro Mode is optional. If you activate it, you support continued development of the project and unlock:

  • Everything that's free, plus...
  • Automatic SCORM patching on download (off by default)
  • Tabbed Pro Code Editor for deeper file management (syntax highlighting as well)
  • Upload your own JS and CSS files
  • Persistent settings and synced local storage
  • Batch processing (coming soon)

Pro licenses are limited to two active devices at a time and validated daily behind the scenes. It’s meant to be simple, not restrictive. There's a free trial, too.

What’s Next

We already have a few features in place that are being polished for release:

  • Live CSS Preview – It works. We're refining the experience before release and intend to make this free.
  • Batch Patching – Apply one theme to up to 10 SCORM files in a single step (Pro).
  • Improved Quick Editor – Small UX touches that make a big difference for quick edits (Free).

Why We’re Doing This

We built RiseOverride because Rise doesn’t offer enough control for people who want to... override things. It’s not personal. We just wanted to write a little code without having to jump through hoops or pay hundreds per year for someone else’s stuff.

This started as a side project between two developer/parent humans. Pro Mode just helps us justify the time. If you think it’s useful, that’s honestly what matters most.

We Want Feedback

We’re listening. Really. If there’s something RiseOverride should do, tell us. If something’s broken, definitely tell us. And if you find a clever use case we haven’t thought of, we’d love to hear about it.

You can install the extension here and share feedback here. We read it all.

Thanks for giving it a try. We’re just getting started.
